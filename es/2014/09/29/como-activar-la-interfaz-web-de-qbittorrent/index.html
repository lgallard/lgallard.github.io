    Cómo activar la interfaz Web de qBittorrent

    less than 1 minute read

    Si deseas controlar to servidor qBittorrent usando la interfaz web, sigue los siguientes pasos:

    1. En la barra de menú, ve a Tools > Options

    2. En la próxima ventana, selecciona la opción Web UI
    3. Marca la opción Enable the Web User Interface (Remote control)
    4. Selecciona un puerto (Por omisión 8080)
    5. Configura el nombre de usuario y contraseña (Por omisión username: admin / password: adminadmin)


    6. Haz clic en Ok para guardar las configuraciones.

    Ahora podrás acceder a tu servidor desde un navegador si colocas la dirección IP y puerto de tu servidor qBittorrent, ejemplo: 192.168.1.100:8080 como se muestra a continuación:


    También puedes acceder desde tu dispositivo Android si instalas qBittorrent Client o qBittorrent Client Pro
